# AWS Storage Gateway

## Qué es
Un servicio que conecta entornos locales con almacenamiento en la nube, proporcionando acceso a datos en AWS.

## Para qué sirve
Proporcionar una forma de integrar el almacenamiento local con AWS, permitiendo el acceso y la gestión de datos en la nube.

## Casos de uso
- Integración de almacenamiento
- Backup de datos
- Recuperación de desastres
- Migración de datos
- Acceso a datos en la nube

## Puntos clave
- **Integración:** Integración de almacenamiento local con AWS
- **Acceso:** Acceso a datos en la nube
- **Caché:** Caché local para acceso rápido
- **Sincronización:** Sincronización de datos entre local y nube
- **Seguridad:** Seguridad y cifrado de datos

## Comparación con AWS Backup
- **AWS Storage Gateway:** Un servicio que conecta entornos locales con almacenamiento en la nube, proporcionando acceso a datos en AWS. Ideal para integración de almacenamiento local con AWS.
- **AWS Backup:** Un servicio que centraliza y automatiza la gestión de copias de seguridad en AWS, proporcionando una solución unificada para backup. Ideal para gestión centralizada de copias de seguridad.
